# 清理缓存并重启服务

## 问题
前端显示异常，可能是浏览器缓存或 Vite 缓存导致的。

## 解决方案

### 1. 停止所有服务
```bash
# 按 Ctrl+C 停止前端和后端服务
```

### 2. 清理 Vite 缓存
```bash
# 删除 node_modules/.vite 缓存目录
rmdir /s /q node_modules\.vite
```

### 3. 重启服务
```bash
# 重启后端
node server/index.js

# 重启前端（新终端）
npm run dev
```

### 4. 清理浏览器缓存
在浏览器中：
1. 按 `Ctrl + Shift + Delete` 打开清除浏览器数据
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

或者：
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 5. 强制刷新页面
```
Ctrl + F5 (Windows)
Cmd + Shift + R (Mac)
```

## 快速命令（Windows）

```cmd
# 一键清理并重启
rmdir /s /q node_modules\.vite && npm run dev
```

## 验证修复

访问以下页面确认功能正常：
1. `/payment-system?tab=wallets` - 代付系统钱包管理
2. `/wallet?tab=wallets` - 钱包配置页面
3. `/wallet?tab=basic&subtab=api` - API配置界面

检查浏览器控制台（F12）的错误信息，并提供给我。
