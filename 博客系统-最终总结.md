### API 文档
**公开接口**：
- `GET /api/blog` - 获取文章列表
- `GET /api/blog/:slug` - 获取文章详情
- `POST /api/blog/:slug/view` - 增加阅读量
- `GET /api/blog/categories` - 获取分类列表

**管理员接口**：
- `POST /api/blog` - 创建文章
- `PUT /api/blog/:id` - 更新文章
- `DELETE /api/blog/:id` - 删除文章
- `POST /api/blog/categories` - 创建分类
- `PUT /api/blog/categories/:id` - 更新分类
- `DELETE /api/blog/categories/:id` - 删除分类
