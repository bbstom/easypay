# 警告弹窗 - 后台设置示例内容

## 使用方法

1. 登录后台管理系统
2. 进入"设置" → "警告弹窗"
3. 启用"启用警告弹窗"开关
4. 填写标题和内容
5. 点击"保存设置"

## 示例 1：防诈骗提示（推荐）

### 标题
```
交易风险重要提示
```

### 内容（HTML）
```html
<div class="space-y-4">
  <p class="font-medium text-gray-900">
    <span class="underline decoration-amber-500 decoration-2">本平台仅提供代付技术服务，并非商品销售商家。</span>
  </p>
  
  <p>
    我们的职责仅限于确保 <span class="font-bold text-blue-600">USDT</span> 准确到达您指定的区块链收款地址。
  </p>
  
  <div class="bg-amber-50 p-4 rounded-lg border-l-4 border-amber-500">
    <p class="text-sm text-amber-800">
      <strong>防诈骗提醒：</strong><br>
      若您是在购买货品，可能存在被骗风险。平台无法强制商家发货，请您务必自行甄别商家信誉。
    </p>
  </div>
  
  <p class="text-red-600 font-bold text-center pulse-warning">
    ⚠️ 请谨慎付款，谨防电信诈骗！
  </p>
</div>
```

## HTML 样式说明

### 可用的 Tailwind CSS 类

#### 文字颜色
- `text-gray-600` - 灰色文字
- `text-blue-600` - 蓝色文字
- `text-red-600` - 红色文字
- `text-amber-800` - 琥珀色文字
- `text-green-800` - 绿色文字

#### 背景颜色
- `bg-gray-50` - 浅灰色背景
- `bg-blue-50` - 浅蓝色背景
- `bg-red-50` - 浅红色背景
- `bg-amber-50` - 浅琥珀色背景
- `bg-green-50` - 浅绿色背景

#### 字体样式
- `font-bold` - 加粗
- `font-medium` - 中等粗细
- `text-sm` - 小号文字
- `text-lg` - 大号文字
- `text-xl` - 超大号文字

#### 装饰效果
- `underline` - 下划线
- `decoration-amber-500` - 琥珀色下划线
- `decoration-2` - 下划线粗细
- `rounded-lg` - 圆角
- `border-l-4` - 左边框
- `shadow-lg` - 阴影

#### 动画效果
- `pulse-warning` - 脉冲动画（已内置）

#### 间距
- `space-y-4` - 垂直间距
- `p-4` - 内边距
- `mb-2` - 下边距
- `mt-1` - 上边距

### 特殊效果

#### 脉冲文字（闪烁提示）
```html
<p class="pulse-warning text-red-600 font-bold">
  ⚠️ 重要提示
</p>
```

#### 高亮框
```html
<div class="bg-amber-50 p-4 rounded-lg border-l-4 border-amber-500">
  <p class="text-amber-800">提示内容</p>
</div>
```

#### 居中大标题
```html
<p class="text-center text-2xl font-bold text-gray-900">
  标题文字
</p>
```

## 测试建议

1. **内容长度测试**
   - 测试短内容（1-2 行）
   - 测试中等内容（3-5 段）
   - 测试长内容（超过屏幕高度）

2. **样式测试**
   - 测试不同颜色组合
   - 测试字体大小
   - 测试边框和背景

3. **响应式测试**
   - 在电脑上查看
   - 在手机上查看
   - 在平板上查看

## 注意事项

1. **HTML 安全**
   - 不要使用 `<script>` 标签
   - 不要使用 `onclick` 等事件
   - 只使用样式类和基本 HTML 标签

2. **内容建议**
   - 保持简洁明了
   - 突出重点信息
   - 使用视觉层次
   - 避免过长的文字

3. **颜色使用**
   - 红色：警告、危险
   - 黄色/琥珀色：提示、注意
   - 蓝色：信息、说明
   - 绿色：安全、建议
