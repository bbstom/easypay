## ⚠️ 注意事项

### 不要修改的内容

以下内容**不应该修改**，因为它们是系统路径、数据库名称或示例数据：

#### 1. 数据库名称
```javascript
mongodb://localhost:27017/easypay  // 保持不变
mongodb://localhost:27017/fastpay  // 保持不变
```

**原因**：修改数据库名称会导致数据丢失，需要迁移所有数据。

#### 2. PM2 进程名称
```javascript
name: 'easypay-backend'  // 保持不变
```

**原因**：修改进程名称需要重新配置 PM2，可能影响监控和日志。

#### 3. 文件路径
```bash
/www/wwwroot/kk.vpno.eu.org/easypay  // 保持不变
```

**原因**：修改路径需要重新部署，可能影响 Nginx 配置。

#### 4. Git 仓库名称
```bash
git clone <repo>/easypay.git  // 保持不变
```

**原因**：修改仓库名称需要更新所有克隆，可能影响 CI/CD。

#### 5. 示例邮箱
```javascript
email: 'admin@fastpay.com'  // 示例数据，保持不变
```

**原因**：这是脚本中的示例数据，不是实际使用的邮箱。
