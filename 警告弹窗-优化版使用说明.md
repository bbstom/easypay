## 📝 使用步骤

### 步骤 1：重启服务器
```bash
# 如果使用 PM2
pm2 restart easypay

# 或者直接重启
npm run dev
```

### 步骤 2：后台配置

1. 登录后台管理系统
2. 进入"设置" → "警告弹窗"
3. 启用"启用警告弹窗"开关
4. 填写标题：`交易风险重要提示`
5. 填写内容（复制下面的 HTML）：

```html
<div class="space-y-4">
  <p class="font-medium text-gray-900">
    <span class="underline decoration-amber-500 decoration-2">本平台仅提供代付技术服务，并非商品销售商家。</span>
  </p>
  
  <p>
    我们的职责仅限于确保 <span class="font-bold text-blue-600">USDT</span> 准确到达您指定的区块链收款地址。
  </p>
  
  <div class="bg-amber-50 p-4 rounded-lg border-l-4 border-amber-500">
    <p class="text-sm text-amber-800">
      <strong>防诈骗提醒：</strong><br>
      若您是在购买货品，可能存在被骗风险。平台无法强制商家发货，请您务必自行甄别商家信誉。
    </p>
  </div>
  
  <p class="text-red-600 font-bold text-center pulse-warning">
    ⚠️ 请谨慎付款，谨防电信诈骗！
  </p>
</div>
```

6. 点击"保存设置"

### 步骤 3：清空测试数据
在浏览器控制台执行：
```javascript
localStorage.removeItem('workspaceAlertDontShowDate');
```

### 步骤 4：测试弹窗
1. 访问 USDT 代付页面：`/pay`
2. 访问 TRX 代付页面：`/pay-trx`
3. 应该能看到优化后的黄色警示弹窗

## 🔧 自定义选项

### 修改主色调
如果想改变黄色为其他颜色，修改以下类：

```jsx
// 将 amber-500 改为其他颜色
bg-amber-500  → bg-red-500   (红色)
bg-amber-500  → bg-blue-500  (蓝色)
bg-amber-500  → bg-green-500 (绿色)
```

### 修改按钮文字
```jsx
我知道了，继续操作  → 确认并继续
我知道了，继续操作  → 我已知晓风险
我知道了，继续操作  → 同意并继续
```

### 修改品牌标识
```jsx
Security Protocol Powered by KSDN
→ 改为你的品牌名称
```

- 不同颜色的提示框
- 图标和 emoji

## 🐛 故障排除

### 问题 1：弹窗不显示
**解决方案：**
1. 检查后台是否启用了弹窗
2. 检查是否填写了内容
3. 清空 localStorage
4. 重启服务器

### 问题 2：样式不正确
**解决方案：**
1. 检查 HTML 语法是否正确
2. 检查 Tailwind 类名是否拼写正确
3. 清空浏览器缓存

### 问题 3：动画不流畅
**解决方案：**
1. 检查浏览器是否支持 CSS 动画
2. 尝试禁用浏览器扩展
3. 更新浏览器到最新版本

### 问题 4：移动端显示异常
**解决方案：**
1. 检查 viewport meta 标签
2. 测试不同的移动设备
3. 使用浏览器开发者工具模拟
