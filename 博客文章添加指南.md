## 为什么会出现"堆叠"问题？

### 原因分析

当你直接复制包含元信息的文件内容时：

```markdown
# USDT 代付完全指南：从入门到精通

## 文章信息
- **标题**：USDT 代付完全指南：从入门到精通
- **Slug**：usdt-payment-complete-guide
...

## 什么是 USDT 代付？
...
```

ReactMarkdown 会将所有内容都渲染出来，包括：
- 文件标题（H1）
- 元信息列表
- 分隔线
- 正文内容

这样就会显得"堆叠"，没有排版。

### 正确的做法

只复制正文内容（从第一个二级标题开始）：

```markdown
## 什么是 USDT 代付？

USDT 代付是一种区块链支付服务...
```

这样 ReactMarkdown 才能正确渲染出美观的排版。

## 常见问题

### Q1：为什么我的表格没有样式？

**原因**：可能缺少 `remark-gfm` 插件。

**解决**：
```bash
npm install remark-gfm
```

### Q2：为什么代码块没有高亮？

**原因**：ReactMarkdown 默认不支持语法高亮。

**解决**：需要安装 `react-syntax-highlighter`（可选）。

### Q3：为什么图片太大？

**原因**：没有设置图片最大尺寸。

**解决**：在后台"系统设置"中配置图片最大宽度和高度。

### Q4：如何添加封面图？

**方式 1**：在后台上传图片
**方式 2**：使用外部图片 URL
**方式 3**：使用默认图片
